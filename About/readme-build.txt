BUILD INSTRUCTIONS FOR SOLARMAZE

To build SolarMaze, navigate to the directory "Build/ou-cs-cg/"
under the root directory of the project.  Here run "Gradle installDist".

To run the program after you have built it, navigate to new "build" directory
under "ou-cs-cg" and continue to "install/SolarMaze/bin/" here you will find
Unix and Windows scripts that will run the program.  Run the system
appropriate script and enjoy!
